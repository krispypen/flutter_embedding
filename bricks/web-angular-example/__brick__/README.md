# angular-web

This Angular project is an example of how to integrate the flutter_embedding app using js interop.

This project was created with Angular CLI.

### Building the app

In order to build this app, first fetch its `npm` dependencies:

```console
$ npm install
```

Then run the `build` script. It'll take care of building Flutter automatically:

```console
$ npm run build
```

Then you can run the app:

```console
$ npm start
```
